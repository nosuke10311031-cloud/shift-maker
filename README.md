# シフトメーカー

バイトのシフトを自動で作って共有する、サーバー不要のツールです。

## 公開されるのは `docs` フォルダの中身

| ファイル | 役割 |
|---|---|
| `docs/index.html` | 入口（3つのボタン） |
| `docs/staff.html` | バイトが希望を入力する画面 |
| `docs/admin.html` | 管理者がシフトを作る画面 |
| `docs/view.html`  | できたシフトを見る画面（閲覧専用） |

**コードを直すときは、この `docs` フォルダの中のファイルを編集します。**

## 公開のしかた

GitHub Pages（GitHubの無料ホームページ機能）で公開します。
手順は [公開手順.md](公開手順.md) を見てください。

## その他のファイル

- `archive/` … 開発途中の古いファイル（もう使いません）
- `HANDOFF.md` … これまでの開発メモ
